## [0.1.0] - 2026-05-18

### Features
- **exception-hierarchy:** Add typed exception hierarchy for API errors
- **http-client-core:** Add authenticated HTTP client with exception mapping
- **card-and-pack-resources:** Add typed readonly DTOs for cards and packs
- **list-packs:** Add packs() and pack() endpoints with fromArray hydration
- **list-cards:** Add cards() and card() endpoints with fromArray hydration
- **card-filter:** Add fluent CardFilter builder and update cards() signature
- **laravel-integration:** Add ServiceProvider and Facade for Laravel integration
- **configurable-base-uri:** Make base URI configurable via constructor parameter
- **remove-default-base-uri:** Make baseUri a required constructor parameter
- **remove-default-base-uri:** Throw when OPCARDS_BASE_URI is not set
